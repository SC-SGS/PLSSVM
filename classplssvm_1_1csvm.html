◆ learn()

template<typename T >
void plssvm::csvm< T >::learn ( )

Learns the support vectors given the data in the provided parameter class.

Performs 2 steps:

  1. Load the data onto the used device (e.g. one or more GPUs)
  2. Learn the model by solving a minimization problem using the Conjugated Gradients algorithm
Exceptions
plssvm::exceptionif no labels are given for training
plssvm::exceptionif the number of labels and number of data points mismatch
