std::vector<command_queue> plssvm::opencl::detail::get_command_queues ( target_platform  target)

Returns the list devices matching the target platform target.

If the selected target platform is plssvm::target_platform::automatic the selector tries to find devices in the following order:

  1. NVIDIA GPUs
  2. AMD GPUs
  3. Intel GPUs
  4. CPUs
