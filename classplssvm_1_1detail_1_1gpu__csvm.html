Detailed Description

template<typename T, typename device_ptr_t, typename queue_t>
class plssvm::detail::gpu_csvm< T, device_ptr_t, queue_t >

A C-SVM implementation for all GPU backends to reduce code duplication.

Implements all virtual functions defined in plssvm::csvm. The GPU backends only have to implement the actual kernel launches.

Template Parameters
Tthe type of the data
device_ptr_tthe type of the device pointer (dependent on the used backend)
queue_tthe type of the device queue (dependent on the used backend)

Learns the support vectors given the data in the provided parameter class.

  2. Learn the model by solving a minimization problem using the Conjugated Gradients algorithm
